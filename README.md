## What Is This?

The CarMax Manager Agent is your AI-powered leadership development platform. It combines:

1. **Document Generation System** (`carmax_idp_system/`) — Generates recognitions, training texts, IDP reports, and leadership communications aligned with CarMax values and vocabulary
2. **CarMax Manager Persona** (`AGENTS.md`) — The institutional identity that any AI assistant can adopt to coach/manage with authentic CarMax voice
3. **Knowledge Base** — 35+ documents including performance reviews, assessments, recognitions, training records, and CarMax reference materials

## Quick Start

### Use the AI Agent
Simply ask your AI assistant (Claude, ChatGPT, Hermes) natural language questions:
- *"Write a recognition for Carlos Barrios — Win Together value"*
- *"Generate my Q2 2026 IDP report targeting Logistics Coordinator"*
- *"Draft a safety alert about tie-down procedures"*
- *"Coach me on SBI-R feedback for a difficult conversation"*
- *"Help me prepare a STAR story for my ADP assessment"*

The Assistant will adopt the CarMax Manager persona automatically.

### Use the CLI
```bash
cd carmax_idp_system/tools
python carmax_cli.py status
python carmax_cli.py recognition --to "Name" --value "Win Together" --reason "helped with delivery"
python carmax_cli.py training --subject "DOT Compliance" --audience "All Drivers"
python carmax_cli.py idp --period "Q2 2026" --target-role "Logistics Coordinator"
python carmax_cli.py communication --type huddle --topic "Morning Safety Brief"
```

## Communication Standards

- **Tone:** Calm • Clear • Positive • Constructive
- **Language:** Inclusive (we/our/team), appreciation-focused, values-aligned
- **Structure:** 5-Step format (Greeting, Purpose, Key Points, Value Connection, Positive Close)
- **Method:** SBI-R (Situation, Behavior, Impact, Recommendation)
