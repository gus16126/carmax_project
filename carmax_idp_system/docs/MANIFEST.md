## 🧾 Recognition Data Handling Policy

**Primary Records:**  
- `recognitions_given_2025-11-04.txt`  
- `recognitions_received_2025-11-04.txt`

These files contain all authentic Achievers recognitions (given and received) and represent the **official source of record** for recognition activity under the CarMax Project.

**Data Retention Rule:**  
- Do not edit or split master recognition files manually.  
- Maintain the original Achievers text formatting and date alignment.  
- Future recognions should be appended to these master files in chronological order, with date and recipient clearly marked.

**Optional Automation:**  
A tool exists to automatically generate individual `.txt` files from each recognition entry for analytics, audits, or report generation.  
This process does **not** modify the master files — it only creates read-only duplicates under derived folders.
